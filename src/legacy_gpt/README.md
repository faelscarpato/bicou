# Referência GPT

Esta pasta registra a área temporária pedida para a integração com o protótipo GPT.

Os arquivos originais consultados foram:

- `src/components/WalletSummary.tsx`
- `src/pages/worker/Wallet.tsx`
- `src/components/ProgressStepper.tsx`
- `src/components/Chip.tsx`
- `src/components/BottomNav.tsx`

As ideias foram reimplementadas nos padrões do Lovable (`shadcn/ui`, `Tailwind`, `lucide-react` e `zustand persist`) para evitar componentes duplicados como `PrimaryButton`, `Card` e `Chip`.
